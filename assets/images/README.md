This folder contains logo assets for use within the app (splash screens, login screens, etc.).

## Required Files:

### 1. Logo Files

#### logo_ar.png (Recommended: 512x512px or higher)
- **Arabic version** of the Wasslni Plus logo
- Text: "وصلني بلس" (Wasslni Plus in Arabic)
- Include both icon and text
- Transparent background
- Use for: Splash screen, login page, headers

#### logo_en.png (Recommended: 512x512px or higher)  
- **English version** of the Wasslni Plus logo
- Text: "Wasslni Plus"
- Include both icon and text
- Transparent background
- Use for: Splash screen, login page, headers

#### logo_icon_only.png (512x512px)
- Just the icon/symbol without text
- Transparent background
- Use for: Small spaces, loading screens

## Design Guidelines:

### Typography:
- Use modern, clean fonts
- Arabic: Use fonts like Cairo, Tajawal, Almarai
- English: Use fonts like Poppins, Inter, Montserrat

### Color Usage:
- **Primary**: #00BCD4 (Teal/Cyan) - 60% of logo
- **Accent**: #FF9800 (Orange) - 30% of logo
- **Neutral**: Black/White - 10% of logo

### Style:
- Modern and clean
- Flat design or subtle gradients
- Avoid overly complex details
- Should work in both light and dark modes

## Brand Personality:
- **Fast** - Dynamic, energetic
- **Reliable** - Professional, trustworthy
- **Modern** - Tech-savvy, contemporary
- **Accessible** - Friendly, easy to use

## File Formats:
- PNG with transparency (primary)
- SVG (optional, for scalability)
- Different sizes for different use cases

## Once Created:
Update the relevant app files to use these logos:
- Splash screen configuration
- Login/registration pages
- App headers
- About page
- Marketing materials
