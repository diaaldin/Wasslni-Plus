## üî§ Typography

### Recommended Fonts

#### For Arabic Text
1. **Cairo** (Primary)
   - Modern, highly legible
   - Good kerning and spacing
   - Weights: Light (300), Regular (400), SemiBold (600), Bold (700)

2. **Tajawal** (Alternative)
   - Clean and professional
   - Good for headlines

3. **Almarai** (Alternative)
   - Excellent readability

#### For English Text
1. **Inter** (Primary)
   - Modern sans-serif
   - Excellent readability
   - Weights: Light (300), Regular (400), Medium (500), SemiBold (600), Bold (700)

2. **Poppins** (Alternative)
   - Friendly, geometric

3. **Montserrat** (Alternative)
   - Clean and modern

### Text Styles

#### Headings
- **H1**: 24px, Bold (700), Primary Text Color
- **H2**: 20px, SemiBold (600), Primary Text Color
- **H3**: 18px, SemiBold (600), Primary Text Color

#### Body Text
- **Body Large**: 16px, Regular (400), Primary Text Color
- **Body Medium**: 14px, Regular (400), Primary Text Color
- **Body Small**: 12px, Regular (400), Secondary Text Color

#### Buttons
- **Button Text**: 16px, Medium (500), Uppercase (optional), Letter spacing: 0.5px

## üéØ Logo & Icon Design

### Logo Specifications

#### Logomark (Icon Only)
- **Size**: 512x512px minimum
- **Style**: Modern, minimalist
- **Elements**: Package/box with speed indicator
- **Colors**: Primary (#1976D2) and Secondary (#03A9F4)

#### Logotype (Text + Icon)
- **Arabic Version**: "ŸàÿµŸÑŸÜŸä ÿ®ŸÑÿ≥" + Icon
- **English Version**: "Wasslni Plus" + Icon
- **Layout**: Icon on left, text on right (RTL for Arabic)
- **Minimum Size**: 120px height

#### App Icon Requirements
- **Size**: 1024x1024px (master file)
- **Format**: PNG with transparency
- **Safe Zone**: Keep important elements within center 80%
- **Android Adaptive**: Provide separate foreground and background
- **iOS**: Remove alpha channel (or provide solid background version)

### Icon Design Concepts

Choose one of these concepts or create a hybrid:

1. **Package with Arrow**
   - Stylized box/package
   - Speed arrow integrated into design
   - Gradient from deep blue to sky blue

2. **Abstract "W+" Letter mark**
   - Modern typography
   - Incorporate package/delivery elements
   - Bold and memorable

3. **Delivery Truck Minimal**
   - Side view of delivery vehicle
   - Motion lines for speed
   - Rounded, friendly style

4. **Pin + Package**
   - Location pin containing a package
   - Represents delivery to location
   - Clean, two-color design

## üìê Design Principles

### 1. Clarity & Simplicity
- Clean layouts with ample white space
- Clear visual hierarchy
- Minimal but meaningful icons

### 2. Speed & Responsiveness
- Fast-loading screens
- Smooth animations (200-300ms)
- Instant feedback for user actions

### 3. Accessibility
- High contrast ratios (4.5:1 minimum for text)
- Touch targets minimum 44x44px
- Support for screen readers
- Scalable text sizes

### 4. Consistency
- Reuse components across screens
- Consistent spacing (8px grid system)
- Unified interaction patterns

### 5. Bilingual Support
- RTL support for Arabic
- LTR support for English
- Mirrored layouts where appropriate
- Language-appropriate typography

## üé¨ Animations & Transitions

### Duration
- **Fast**: 150ms - Hover states, button presses
- **Normal**: 250ms - Component transitions, modal open/close
- **Slow**: 400ms - Page transitions, major state changes

### Easing
- **Standard**: cubic-bezier(0.4, 0.0, 0.2, 1)
- **Deceleration**: cubic-bezier(0.0, 0.0, 0.2, 1)
- **Acceleration**: cubic-bezier(0.4, 0.0, 1, 1)

### Common Animations
- Fade in/out for modals
- Slide up for bottom sheets
- Slide left/right for page transitions (respect RTL)
- Pulse for loading states
- Bounce for success confirmations
